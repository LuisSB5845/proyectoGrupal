# Sistema de Gestión de Clínica

Una aplicación de escritorio nativa desarrollada en Java para la administración y control interno de una clínica médica. Este sistema permite gestionar pacientes, historiales y procesos administrativos de manera eficiente y centralizada.

## 📌 Sobre el Proyecto

Este software fue desarrollado como proyecto académico para aplicar los pilares de la **Programación Orientada a Objetos (POO)** en un entorno práctico. Demuestra el uso de interfaces gráficas estructuradas, manejo de eventos, y persistencia de datos mediante serialización binaria local.

## ✨ Características Principales
- Registro y mantenimiento de datos de pacientes.
- Gestión de expedientes médicos.
- Interfaz gráfica de usuario (GUI) amigable e intuitiva.
- Almacenamiento local persistente sin dependencia de un motor de base de datos externo.

## 🛠 Tecnologías Utilizadas
- **Lenguaje:** Java
- **Interfaz Gráfica:** Java Swing / AWT
- **Librería UI:** JGoodies Forms (para estructuración avanzada de layouts)
- **Entorno de Desarrollo:** Eclipse IDE

## 🚀 Cómo ejecutar el proyecto

Este proyecto está estructurado como un espacio de trabajo nativo de Eclipse (`.project` y `.classpath`).

1. Abre **Eclipse IDE**.
2. Ve a `File > Import > General > Existing Projects into Workspace`.
3. Selecciona la carpeta `ProyectoClinica` de este repositorio.
4. Asegúrate de que la librería externa `jgoodies-forms-1.8.0.jar` (incluida en el repositorio) esté correctamente enlazada en el *Build Path* del proyecto.
5. Ejecuta la clase principal (`Main`) para iniciar la aplicación.
